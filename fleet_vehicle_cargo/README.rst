===========================
Vehicle Cargo Specification
===========================

This module add vehicle's cargo specifications info, such as:

1. Floor Space
2. Loading Space
3. Load Capacity

Installation
============

To install this module, you need to:

1.  Clone the branch 8.0 of the repository https://github.com/open-synergy/opensynid-fleet
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Setting -> Modules -> Local Modules*
5.  Search For *Vehicle Cargo Specifications*
6.  Install the module

Usage
=====

1. Go to *Fleet -> Vehicle -> Vehicle*
2. Create/open vehicle data
3. On vehicle data, activate *Cargo* flag
4. Update cargo specification
